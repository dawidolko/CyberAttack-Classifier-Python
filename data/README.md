The file `cybersecurity_attacks.csv` is **not** committed to Git (size). It is downloaded automatically on first run via `download_data.py` or when you run `pipeline.py` / `start.sh` / `start.bat`.

Source: [Cyber Security Attacks (Kaggle)](https://www.kaggle.com/datasets/teamincribo/cyber-security-attacks).

To fetch manually: `python download_data.py` (from the project root, with the virtual environment activated).
